# WeekFlow

A personal activity tracker and AI-powered weekly performance app built with React Native + Expo.

Plan your week, log daily life activities, track focus time, and receive an AI-generated performance report every week — graded, analysed, and personalised to your actual data.

---

## Features

- **Weekly Task Planner** — Add tasks with category, priority, and time estimates. Tick them off as you go.
- **Daily Journal** — Log anything each day: movies watched, workouts, social events, gaming sessions. The AI reads these when generating your report.
- **Focus Time Logging** — Manual entry or live timer. Tracks deep work hours across the week.
- **Energy & Mood Check-in** — Rate your energy (1–5) at the start and end of each week.
- **Weekly Intention** — One sentence goal that the AI references in your report.
- **AI Performance Report** — End-of-week grade (S/A/B/C/D), score out of 100, wins, improvements, capacity insights, priority analysis, and a motivational note — all generated by Claude.
- **Carry-Over System** — Unfinished tasks from last week auto-suggested for the new week with one tap.
- **Streak Tracking** — Weeks in a row hitting 80%+ task completion.
- **History** — Every past week's report, tasks, and journal entries, with grade and date filters.
- **Dashboard** — Trend charts for weekly score, completion rate, tasks by category, and all-time category split.

## Tech Stack

| Layer | Technology |
|---|---|
| Mobile | React Native + Expo SDK 56 |
| Language | TypeScript (strict mode) |
| Navigation | Expo Router |
| Database | Supabase (PostgreSQL + Auth) |
| AI Reports | Anthropic Claude API via Supabase Edge Function |
| State | Zustand + Immer |
| Charts | Victory |
| Animations | React Native Reanimated |

---

## Architecture

- All AI calls go through a **Supabase Edge Function** — the Anthropic API key is never exposed client-side
- Every table has **Row Level Security** scoped to `auth.uid() = user_id` — multi-user ready from day one
- Zustand stores use **optimistic updates** with offline queue fallback via SecureStore
- Weeks use a **composite primary key** `(user_id, id)` to support multiple users on the same week date

## Getting Started

```bash
# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Add your Supabase URL and anon key to .env

# Start development server
npx expo start --web

# Run tests
npx jest
```

For the AI reports to work, deploy the Edge Function and set your Anthropic API key as a Supabase secret:

```bash
supabase functions deploy generate-report
supabase secrets set ANTHROPIC_API_KEY=your-key-here
```
